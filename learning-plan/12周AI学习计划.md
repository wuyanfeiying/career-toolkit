#### Day 1 代码（直接复制到 [Colab](https://colab.research.google.com/)）

```python
# ===== Day 1：跑通文本生成 =====
!pip install transformers -q

from transformers import pipeline

gen = pipeline("text-generation", model="gpt2")

prompts = [
    "Transformer is a model that",
    "The key idea of Attention is",
    "In deep learning, we",
]
for p in prompts:
    result = gen(p, max_new_tokens=60, do_sample=True, temperature=0.8)
    print(f">>> {p}")
    print(result[0]["generated_text"])
    print()
```

#### Day 2 代码（可视化 Attention）

```python
# ===== Day 2：可视化哪些词在"关注"哪些词 =====
!pip install transformers bertviz -q

from transformers import AutoTokenizer, AutoModel
from bertviz import head_view

model_name = "bert-base-uncased"
tokenizer = AutoTokenizer.from_pretrained(model_name)
model = AutoModel.from_pretrained(model_name, output_attentions=True)

sentence = "The cat sat on the mat because it was tired"
inputs = tokenizer(sentence, return_tensors="pt")
outputs = model(**inputs)

tokens = tokenizer.convert_ids_to_tokens(inputs["input_ids"][0])
head_view(outputs.attentions, tokens)
# 交互图：每个色块=一个注意力头，颜色越深=关注度越高
# 截图备用，视频里可以直接展示这张图
```
